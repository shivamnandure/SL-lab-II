## For VS Code Terminal / Command Prompt

Use this command:

```bash id="btv2t7"
pip install pandas matplotlib seaborn scikit-learn notebook
```

### Used For

* VS Code Terminal
* Windows CMD
* PowerShell
* Git Bash

---

## For Anaconda Package Manager

Use this command:

```bash id="s8xz8g"
conda install pandas matplotlib seaborn scikit-learn notebook
```

### Used For

* Anaconda Prompt
* Jupyter Notebook via Anaconda

---

## Optional: Launch Jupyter Notebook

After installation:

```bash id="h1gxee"
jupyter notebook
```

This opens Jupyter Notebook in your browser for running the `.ipynb` file.
