# 📌 Pre-Installation Commands for This Practical

This practical uses:

* Pandas
* Matplotlib
* Seaborn
* Scikit-learn
* NLTK

---

# ✅ 1. VS Code Terminal / CMD Commands

Use these commands in:

* VS Code Terminal
* Windows CMD
* PowerShell

## 🔹 Install Required Libraries

```bash
pip install pandas matplotlib seaborn scikit-learn nltk
```

---

## 🔹 Verify Installation

```bash
pip list
```

---

## 🔹 Run Python File

```bash
python filename.py
```

Example:

```bash
python sentiment_analysis.py
```

---

# ✅ 2. Anaconda Prompt Commands

Use these commands in:

* Anaconda Prompt

## 🔹 Install Required Packages

```bash
conda install pandas matplotlib seaborn scikit-learn nltk
```

---

## 🔹 If Some Packages Are Missing

Use pip inside Anaconda:

```bash
pip install nltk
```

---

## 🔹 Run Python File

```bash
python filename.py
```

---

# 📌 Important First-Time NLTK Setup

Run this once in Python:

```python
import nltk
nltk.download('stopwords')
```

This downloads required NLP data.

# 📌 Recommended Order During Practical Exam

## If Using VS Code

1. Install libraries
2. Run program
3. Show graphs
4. Test user input

---

## If Using Jupyter Notebook

1. Open notebook
2. Run cells one-by-one using Shift + Enter
3. Show outputs and visualizations
