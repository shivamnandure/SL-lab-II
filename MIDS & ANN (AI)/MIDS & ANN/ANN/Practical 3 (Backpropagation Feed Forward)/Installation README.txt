Before running this practical, install the required Python libraries using this command in Anaconda Prompt or terminal:

```bash
pip install numpy matplotlib
```

---

# How to Run the Installation Command

## In Anaconda

1. Open **Anaconda Navigator**
2. Launch **Anaconda Prompt**
3. Type:

```bash
pip install numpy matplotlib
```

4. Press **Enter**
5. Wait until installation completes

---

# What These Libraries Do

| Library    | Purpose                                         |
| ---------- | ----------------------------------------------- |
| NumPy      | Mathematical calculations and matrix operations |
| Matplotlib | Graph plotting and visualization                |
