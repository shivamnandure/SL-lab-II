Before running this ART1 practical in Anaconda/Jupyter, install the required libraries using this command:

```bash id="nd9f0f"
pip install numpy matplotlib
```

If you are using **Anaconda Prompt**, you can also use:

```bash id="oe6r0m"
conda install numpy matplotlib
```

---

# What These Libraries Do

* NumPy
  → Used for array operations and similarity calculations

* Matplotlib
  → Used for graph visualization
