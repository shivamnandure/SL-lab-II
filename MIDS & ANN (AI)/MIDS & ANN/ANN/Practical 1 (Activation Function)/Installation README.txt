Before running this practical, install the required libraries using these commands in **Anaconda Prompt**, **Command Prompt**, or **Terminal**.

### Required Installation Commands

```bash
pip install numpy
pip install matplotlib
```

Or install both together:

```bash
pip install numpy matplotlib
```

### If You Are Using Anaconda

You can also use:

```bash
conda install numpy
conda install matplotlib
```

Or together:

```bash
conda install numpy matplotlib
```

### Quick Viva Explanation

* **NumPy** → Used for mathematical operations and arrays
* **Matplotlib** → Used for graph plotting and visualization

### How to Open Anaconda Prompt

1. Open **Start Menu**
2. Search **Anaconda Prompt**
3. Open it
4. Type the installation command
5. Press **Enter**

After installation, your program will run successfully.
