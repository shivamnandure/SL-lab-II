Before running this practical, install the required libraries using this command in **Anaconda Prompt**, **CMD**, or **Terminal**:

```bash id="b51y4v"

```


3. Open it
4. Paste the command
5. Press Enter

# What These Libraries Do

* `numpy` → Matrix calculations for neural network
* `matplotlib` → Graph visualization for error reduction

That’s all you need for this XOR Back Propagation practical.
