Before running this practical, install the required Python libraries using this command:

```bash id="pre1"
pip install numpy matplotlib
```

If you are using Anaconda Prompt, you can also use:

```bash id="pre2"
conda install numpy matplotlib
```

---

# How to Run the Installation Command

## In Anaconda Prompt

1. Open Anaconda Navigator
2. Click on “Anaconda Prompt”
3. Type the command:

   ```bash
   pip install numpy matplotlib
   ```
4. Press Enter
5. Wait until installation completes

---

# Why These Libraries Are Needed

* `numpy` → Numerical calculations and arrays
* `matplotlib` → Graphical visualization and decision region plotting
